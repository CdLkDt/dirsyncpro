3. Disclaimer of 'Check for updates'
By default (every new version of) DirSync Pro checks for program updates as soon as the program is started (and the license is accepted). Except in the unstable versions, you may disable checking for updates in the Options menu (Tools > Options). When checking for updates, DirSync Pro will send ONLY the current program alphanumeric version indication (for example '1.51' or '1.51b1') to the update server to verify if a newer version is released.
